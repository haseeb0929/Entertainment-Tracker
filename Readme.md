# Entertainment Tracker

## Introduction
Entertainment Tracker is a web-based application for tracking movies, TV shows, and games.  
It solves the problem of managing entertainment lists by allowing users to organize, track, and plan what they want to watch or play.

## Features
- Add/remove entertainment items
- Mark items as Watched, In Progress, or Wishlist
- Search and filter items
- Dashboard for overview of all entertainment

## Tech Stack
- HTML, CSS, JavaScript (or React if using it)
- Node.js / Backend (if applicable)
- MongoDB or SQL database (if applicable)

## How to Run the Project
1. Clone the repository: `git clone <repo-link>`
2. Navigate to project folder
3. For frontend-only: Open `index.html` in browser  
   For fullstack: Run `npm install && npm start` for frontend and backend
4. Ensure the database server is running 

## Results / Key Points
- Helps users track entertainment effectively
- Organized view of all media items
- Easy management with status marking
